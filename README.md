# Objective_1_pipeline

This outlines the process that I used for the Objective 1 analysis for my PhD. 

The steps include: 
- Running Snippy (https://github.com/tseemann/snippy) with gubbins (https://github.com/nickjcroucher/gubbins)
- Determining clades by RheirBAPS (https://github.com/gtonkinhill/rhierbaps)
- Adding metadata to core alignment headers
- Subsetting isolates via the clades
- Running IQTree model selector
- Midpoint rooting the trees
